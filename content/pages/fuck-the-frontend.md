The sentence ships first.

The interface earns its place after that.

Fancy frontend work is not evil. Sometimes it clarifies, compresses, guides, or makes something easier to understand. But decoration is not intelligence. Motion is not meaning. A beautiful shell around a weak idea is still a weak idea.

So this site keeps a small escape hatch: **frontend off**.

Press `f` or use the toggle. The point should survive without the costume.

Rules:

- If styling is louder than the idea, delete the styling.
- If animation does not help comprehension, it is entertainment.
- If the page fails when CSS is stripped, the page was lying about being content.
- The best interface is often a document.
- Polish is allowed. Worship is not.
